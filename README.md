# WebScrapping-Price_Tracker
WebScrapping using BeautifulSoup and tracking product price and sending update to email every 12 hours

## Technologies used
* BeautifulSoup
* Python

## Installation of Python Libraries
* install request
* install bs4
* install time
* install smtplib
